# HEM

## サイト概要
食事内容やレシピを投稿するアプリケーション。
食事内容から不足している栄養素を算出し、不足分を補うレシピを
検索して表示します。

### サイトテーマ
不足している栄養を補うレシピの提供

### テーマを選んだ理由
食事内容を保存する事で、不足している栄養素とその栄養素を含む食材を
利用したレシピを提供するアプリケーションは存在するが、
それでは不足栄養素を取ることはできても、不足分を補うことが出来なかった為。

### ターゲットユーザ
栄養を考えながら献立を作ることに面倒くささを感じている方々。

### 主な利用シーン
献立作成時
